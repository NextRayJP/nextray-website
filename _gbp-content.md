## カテゴリ設定

- **メインカテゴリ（1つのみ）**：`ウェブデザイナー`
  - 「ウェブデザイン会社」も候補ですが、地域の競合状況を見て選んでください
  - メインカテゴリは順位への影響が最も大きい項目です
- **サブカテゴリ（複数可）**：
  - `マーケティング代理店`
  - `広告代理店`
  - `コンピュータサポート、およびサービス`
  - `ビジネス管理コンサルタント`

※ 実際に選べるカテゴリ名はGBPの候補から選択する形式です。上記に完全一致がない場合は
近いものを選んでください。**提供していないサービスのカテゴリは設定しないこと。**

## 口コミ返信テンプレート

**全件に返信してください。** 返信は投稿者だけでなく、それを読む未来のお客様に向けたものです。

### 高評価（★4〜5）への返信

```
〇〇様

このたびはうれしいお言葉をありがとうございます。
「（口コミ内で触れられた具体的な点）」とおっしゃっていただけたこと、
制作を担当した者にも伝えます。大変励みになります。

公開後の運用についても引き続きご一緒させてください。
今後ともよろしくお願いいたします。

NextRay 谷川
```

**ポイント**：口コミ本文の具体的な部分に必ず触れること。定型文だと読み手に伝わります。

### 低評価（★1〜3）への返信

```
〇〇様

このたびはご期待に沿えず、申し訳ございませんでした。
いただいたご指摘について社内で確認し、（改善する内容）を見直しております。

お手数をおかけしますが、詳しい経緯をお聞かせいただけますと幸いです。
080-7517-4514 までご連絡いただければ、私が直接対応いたします。

NextRay 谷川
```

**ポイント**：
- 感情的に反論しない。事実に誤りがある場合も、まず受け止めてから穏やかに補足する
- 公開の場での議論を避け、個別連絡に誘導する
- 「対応しました」で終わらせず、何をどう見直したかを書く

### 内容に心当たりがない口コミへの返信

```
〇〇様

ご投稿ありがとうございます。
恐れ入りますが、いただいた内容について弊社の記録で確認ができておりません。
お心当たりの件がございましたら、080-7517-4514 までご連絡いただけますでしょうか。
状況を確認のうえ、真摯に対応させていただきます。

NextRay 谷川
```

**注意**：明らかな誹謗中傷やポリシー違反の投稿は、Googleに報告できます。
ただし**削除されるとは限りません**。報告と並行して、上記のような冷静な返信を
残しておくほうが、読み手からの信頼は守られます。

## 写真の運用

`assets/gbp/` に以下の画像が用意済みです。

- `gbp-logo.png` — ロゴ
- `gbp-cover.png` — カバー画像
- `gbp-service-hp.png` — ホームページ制作
- `gbp-service-meo.png` — MEO対策
- `gbp-service-ai.png` — AI自動化

**追加したい写真**（あるとプロフィールが充実します）：
- 代表の顔写真（`assets/profile-tanikawa.jpg` が使えます）
- 作業風景・打ち合わせの様子
- 制作したサイトの画面（お客様の許可が取れた場合のみ）

月に数枚ずつ追加を続けるのが理想です。一度にまとめて入れて終わりにしないこと。

## 月次で記録する指標

GBPの「インサイト」で確認できます。スプレッドシート等に月1回記録してください。

| 指標 | 見るポイント |
|---|---|
| 検索された語句 | 想定どおりのキーワードで見つかっているか |
| 表示回数（検索・マップ別） | 増えているか。減った月は何をしたか（しなかったか） |
| ウェブサイトのクリック数 | 表示からサイトへの遷移率 |
| ルート検索 | 実際に来店意図がある人の数 |
| 通話数 | 最終的な成果に最も近い指標 |

**注意**：数字は季節や競合の動きでも変動します。単月で一喜一憂せず、
3ヶ月程度の傾向で判断してください。
